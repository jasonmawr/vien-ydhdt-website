# **PROJECT PRD: WEBSITE VIỆN Y DƯỢC HỌC DÂN TỘC**

## **1\. TỔNG QUAN (OVERVIEW)**

Tái thiết kế hệ thống website của Viện Y Dược Học Dân Tộc chuyển từ WordPress cũ sang kiến trúc hiện đại (Next.js Frontend \+ API Backend).

Mục tiêu: Đẹp, tốc độ tải chớp nhoáng, tối ưu hóa quy trình Đặt lịch khám và Tra cứu thông tin y khoa.

## **3\. CÁC TÍNH NĂNG CỐT LÕI (CORE FEATURES)**

1. **Trang chủ (Homepage):** Banner, Khung tìm kiếm/Tra cứu nhanh, Danh sách Chuyên khoa, Bác sĩ tiêu biểu, Tin tức nổi bật.  
2. **Module Đặt lịch khám:** Biểu mẫu đa bước (Chọn khoa \-\> Chọn Bác sĩ \-\> Chọn Ngày giờ \-\> Điền thông tin).  
3. **Module Từ điển Dược liệu:** Danh sách các vị thuốc Đông y, công dụng, cách dùng (hiển thị dạng lưới/Grid).  
4. **Module Đội ngũ Bác sĩ:** Profile chi tiết của từng y bác sĩ.  
5. **Trang nội dung (Tin tức/Nghiên cứu khoa học):** Chuẩn SEO, tốc độ tải trang nhanh, có mục bài viết liên quan.

## **4\. ĐỐI TƯỢNG NGƯỜI DÙNG (USER PERSONAS)**

* **Người bệnh/Người dân:** Cần tìm thông tin bệnh lý, tra cứu thuốc, đặt lịch khám nhanh chóng trên điện thoại.  
* **Bác sĩ/Sinh viên y khoa:** Tra cứu tài liệu nghiên cứu chuyên sâu, danh mục dược liệu chuẩn.  
* **Quản trị viên (Admin):** Đăng bài viết, quản lý lịch hẹn (Sẽ phát triển ở Phase sau).
